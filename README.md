# Telegram TinderBot

Учебный пет-проект Telegram-бота, разработанный в рамках Python-марафона от JavaRush. Бот имитирует функционал/генерирует контент для приложения знакомств Tinder с использованием OpenAI ChatGPT для генерации контента и взаимодействия.

В перспективе планируется добавить возможность выбора другой языковой модели (например DeepSeek) путем добавления dpsk.py, где будет реализована логика работы с DeepSeek - API

## Возможности

1. **Генерация Tinder-профиля** - создает детализированный профиль на основе предоставленного описания.
2. **Написание сообщений для знакомства** - генерирует интересные и интригующие открывающие сообщения.
3. **Ведение переписки от вашего имени** - автоматически поддерживает диалог, имитируя ваш стиль общения.
4. **Тренировка переписки** - позволяет практиковаться в общении с ИИ-собеседником.
5. **Обычный режим чата с GPT** - отвечает на любые вопросы в формате стандартного диалога с ChatGPT.

## Перспективы развития

- **Поддержка других языковых моделей** - планируется добавить возможность выбора альтернативных LLM (например, DeepSeek) путем создания модуля `dpsk.py` с логикой работы с соответствующим API
- **Анализ эффективности сгенерированных сообщений** - Обратная связь чата на реальные диалоги / еденичные сообщения
- **Мультиязычность** - поддержка различных языков для интерфейса и генерации контента

## Установка и настройка

### Предварительные требования
- Python 3.8 или выше
- Установленный `pip`
- Аккаунт в Telegram (для создания бота через [@BotFather](https://t.me/BotFather))
- API-ключ от OpenAI (можно получить на [platform.openai.com](https://platform.openai.com/))

### Шаги установки

1. **Клонирование репозитория**
   ```bash
   git clone <ссылка-на-репозиторий>
   cd Telegram_TinderBot
   ```

2. **Создание виртуального окружения (если требуется)**
   ```bash
   python -m venv venv

   source venv/bin/activate  # Для Linux/Mac
   venv\Scripts\activate     # Для Windows
   ```

3. **Установка зависимостей**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройка API-ключей**
   Создайте файл `api_keys.py` в корневой директории проекта:
   ```python
   # api_keys.py
   telegram_key = "ВАШ_TELEGRAM_BOT_TOKEN"
   chat_gpt_key = "ВАШ_OPENAI_API_KEY"
   ```

5. **Запуск бота**
   ```bash
   python bot.py
   ```

## Использование

После запуска бота, вы можете взаимодействовать с ним через Telegram. Основные команды доступны через меню бота:

- **/start** - Начать работу с ботом
- **/profile** - Сгенерировать новый Tinder-профиль
- **/opener** - Генерация первого сообщения для знакомства
- **/message** - Получить предложение для первого сообщения
- **/date** - Виртуальная переписка со звездами
- **/gpt** - Перейти в обычный режим ChatGPT

## Описание файлов

### `bot.py`
Содержит основную логику Telegram-бота: обработку команд, управление диалогами и интеграцию с GPT-сервисом.

### `util.py`
Вспомогательные функции для работы с Telegram API:
- Форматирование и отправка сообщений (текст, HTML, с кнопками)
- Работа с меню и командами бота
- Загрузка ресурсов (сообщений, промптов)
- Утилиты для диалогов

### `gpt.py`
Класс `ChatGptService` для взаимодействия с OpenAI API:
- Управление историей сообщений
- Отправка запросов к GPT-3.5-turbo
- Настройка промптов и параметров генерации

### `api_keys.py`
Файл для хранения конфиденциальных данных (API-ключей).

## Конфиденциальность

- Бот не хранит персональные данные пользователей
- История диалогов сохраняется только во время текущей сессии
- Для работы с OpenAI API используется стандартное шифрование
